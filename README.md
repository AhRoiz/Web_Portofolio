# VEE.DEV - Futuristic Personal Portfolio

![Portfolio Preview](https://via.placeholder.com/1200x600.png?text=VEE.DEV+Portfolio+Preview)
*(Catatan: Ganti link gambar di atas dengan screenshot asli website kamu setelah di-deploy)*

Website portofolio pribadi interaktif yang dirancang dengan tema futuristik dan **Cyber/Tech aesthetic**. Dibangun menggunakan **React JS** dan **Tailwind CSS**, website ini menampilkan visualisasi 3D yang memukau, animasi yang halus, dan antarmuka pengguna bergaya *glassmorphism*.

## üåü Fitur Utama

* **Terminal Boot Loading Screen:** Intro unik bergaya "VeeOS v2.5.0" dengan urutan booting sistem sebelum masuk ke halaman utama.
* **3D Interactive Background:** Visualisasi partikel ombak (Waves) dan Aurora menggunakan **Three.js** yang merespons pergerakan mouse dan scroll.
* **Cinematic Project Showcase:** Slider proyek bergaya "queue" dengan modal detail yang imersif untuk menampilkan galeri dan deskripsi teknis.
* **Glassmorphism UI:** Desain modern dengan efek blur dan transparansi yang konsisten di seluruh elemen (Navbar, Cards, Modals).
* **Smooth Scroll Animations:** Elemen muncul secara elegan saat digulir menggunakan *Intersection Observer*.
* **Responsive Design:** Tampilan yang optimal di perangkat desktop, tablet, maupun mobile.

## üöÄ Cara Menjalankan (Local Development)

Ikuti langkah-langkah ini untuk menjalankan proyek di komputer lokalmu:

1.  **Clone Repository**
    ```bash
    git clone [https://github.com/AhRoiz/Web_Portofolio.git](https://github.com/AhRoiz/Web_Portofolio.git)
    cd Web_Portofolio
    ```

2.  **Install Dependencies**
    Pastikan Node.js sudah terinstall.
    ```bash
    npm install
    ```

3.  **Jalankan Server Development**
    ```bash
    npm run dev
    ```

4.  **Buka di Browser**
    Buka `http://localhost:5173` (atau port yang ditampilkan di terminal).
